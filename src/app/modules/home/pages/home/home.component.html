Getting Started

This app was designed to give friendly and useful functions to calculate COCOMOs, generate initial estimates, and generally assist with understanding project effort levels.


Select one of the items below by clicking the button corresponding to the function you would like to start using. You can also use the navigation menu to quickly navigate between functions.

Calculate COCOMO

COCOMO was originally developed by Barry W. Boehm, an american software engineer, to assist with procedurally generating software project estimates.


COCOMO can be a useful first step to understanding effort levels required for software projects.
